# 🖋️ Structure Your Story (`sys`)

My first vibe coded application - Trying this out! It's stable as of now!

**Minimalist. Professional. Zen.**  
A terminal-first screenplay and novel editor designed to help writers focus on structure, not formatting.

---

## ✨ Features

- **CLI-First Workflow**: Start writing instantly with `sys story.json`.
- **Zen Writing Space**: A distraction-free TUI for focused drafting.
- **Professional PDF Export**: Industry-standard PDF output with perfect typography.
- **Structure-Driven**: Templates for Dan Harmon, Save the Cat, and more.
- **Automatic Exports**: Saves directly to your project's directory.

## 🚀 Installation & Download

### 1. Download Pre-compiled Binary (Easiest)
Go to the **[Releases](https://github.com/nkr/StructureYourStory/releases)** page and download the executable for your OS:
- **`sys-linux`**: For Linux (Ubuntu, Debian, etc.)
- **`sys-macos`**: For macOS (Intel/M1/M2)
- **`sys-windows.exe`**: For Windows

*Just download and run! No installation required.*

---

### 2. Manual Installation (For Rust Users)
If you have [Rust](https://www.rust-lang.org/tools/install) installed, run:

```bash
cargo install --path .
```
The app will be available globally as `sys`.

---

## 📖 Usage

### Start or Load a Project
Simply type `sys` followed by a filename:
```bash
sys project.json
```
