A terminal-first screenplay and novel editor designed to help writers focus on structure, not formatting.

## 🚀 Installation (Linux & macOS)

Follow these simple steps to install `sys` on your computer:

### 1. Install Rust
If you don't have Rust yet, open your terminal and run:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
*Follow the on-screen prompts and restart your terminal when done.*

### 2. Install the App
Navigate to the `StructureYourStory` folder and run:
```bash
cargo install --path .
```

### 3. Start Writing
You can now run the app from any folder by typing `sys` followed by a filename:
```bash
sys my_story.json
```

## 📖 How to Use

`sys` is designed to stay out of your way.

1.  **Open or Create**: `sys masterpiece.json`
2.  **Choose a Structure**: If it's a new file, pick a template (like "Save the Cat" or "Dan Harmon").
3.  **Write**: Fill in your story beats.
4.  **Export**: Press `CTRL + X` to generate a professional PDF in the same folder.

### ⌨️ Hotkeys
| Key | Action |
| :--- | :--- |
| **TAB** | Next Beat (Auto-save) |
| **SHIFT + TAB** | Previous Beat |
| **ENTER** | New Line |
| **CTRL + X** | Export Menu |
| **ESC** | Save & Exit |
